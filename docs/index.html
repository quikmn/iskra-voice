Features

Everything you need.
Nothing you don't.

Iskra ships with the full feature set of a modern chat platform — built for people who want to own what they run.

Voice Channels

Push-to-talk or voice activation. WebRTC peer-to-peer with TURN relay for friends behind strict NATs. Low latency, no middleman.

Text Channels

Markdown rendering, image embeds, file uploads, reactions, edits, deletes, pinning, and slow mode. Stored as plain .jsonl files on disk.

Roles & Auth

Five role tiers: guest, member, trusted, admin, owner. Three auth modes: open, registered+guests, or verified-only. HWID-based identity.

Direct Messages

Private one-on-one conversations with read receipts. Works over the same connection as your server — no extra setup.

7 Visual Themes

Catppuccin, Nord, Tokyo Night, Gruvbox, Dracula, and more. Themes sync the Windows title bar color too, not just the web UI.

Soundboard & Screen Share

Play audio clips to your channel, share your screen, or stream your webcam. No extra software, no plugins.

Audit Log & Backup

Every admin action is logged. Download a full server backup (ZIP) from the client — history, roles, bans, avatars, uploads, all of it.

Webhooks & Bots

Post to any text channel via webhook. Build bots that connect as a regular client. No rate limits, no approval process.

System Tray & Native Feel

Minimizes to tray, shows balloon notifications, respects your theme. Feels like a native Windows app because it is one.

Get started

Up in three steps.

No installer, no cloud setup, no account creation. Just unzip and run.

1

Run the server

Extract iskra_server.exe, create a world folder with a server.json, and launch. It's online in seconds.

2

Share the address

Forward one TCP port on your router. Give friends your IP. That's the entire setup — no domain, no SSL cert, no dashboard.

3

Connect & talk

Friends unzip the client, enter your address, pick a name, and they're in. Voice, text, files — everything works immediately.

# Minimal server.json — paste, edit ServerName, run.
{
  "Settings": {
    "ServerName": "My Server",
    "Port": 8080,
    "RequirePassword": false,
    "AdminPassword": "change-this",
    "AuthMode": "open"
  },
  "Channels": [
    { "Id": "v_lobby", "Name": "Lobby", "Type": "Voice" },
    { "Id": "t_gen", "Name": "general", "Type": "Text" }
  ]
}
